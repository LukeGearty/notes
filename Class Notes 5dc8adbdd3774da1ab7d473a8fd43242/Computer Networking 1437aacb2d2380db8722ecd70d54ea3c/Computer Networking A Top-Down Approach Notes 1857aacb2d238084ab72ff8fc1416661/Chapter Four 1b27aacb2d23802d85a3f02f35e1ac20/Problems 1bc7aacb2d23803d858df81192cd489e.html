P2. Suppose two packets arrive to two different input ports of a router at exactly the same time. Also suppose there are no other packets anywhere in the router.

  1. Suppose the two packets are to be forwarded to two different output ports. Is it possible to forward the two packets through the switch fabric at the same time when the fabric uses a shared bus?

    No, since it is shared, then only one packet can go through the fabric at a time.
  1. Suppose the two packets are to be forwarded to two different output ports. Is it possible to forward the two packets through the switch fabric at the same time when the fabric uses switching via memory?

    No, because the read/write operation can only do 1 at a time.
  1. Suppose the two packets are to be forwarded to the same output port. Is it possible to forward the two packets through the switch fabric at the same time when the fabric uses a crossbar?

    No since it is being forwarded to the same output port.

P17.  Suppose datagrams are limited to 1,500 bytes (including header) between
source Host A and destination Host B. Assuming a 20-byte IP header, how
many datagrams would be required to send an MP3 consisting of 5 million
bytes? Explain how you computed your answer.

If the file size is 5,000,000, and assuming there is a TCP header that is also 20-bytes, then that would leave 1460 bytes for the data.

5,000,000 / 1460 = 3425 datagrams
