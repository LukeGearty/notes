Threat Vector

A method used by the attacker to:

Gain access

Also called ‘attack vectors’

Message-based Vectors

One of the biggest and most successful vectors

Email

Malicious link in an email

Link to malicious site

SMS - short message service

attacks in a text message

Phishing attacks

Social engineering attacks

Image-based vectors

Some image formats can be a threat

SVG - image is described in XML and in that XML there can be significant security concerns - HTML injection, Javascript attack code

File-based vectors

Malicious executables

Adobe PDF - a file containing other objects like text or images

ZIP/RAR files or any other compression type - contains many different files

Voice Call Vectors

Vishing

Spam over IP

War Dialing - attackers try to find unpublished phone numbers

Call Tampering

Removable Device Vectors

Get around the firewall through the USB interface

Infect air-gapped networks (no direct network connection to that network)

USB devices can act as keyboards

Data Exfiltration

Vulnerable software Vectors

Client-based

Infected executable, known or unknown vulnerabilities, may require constant updates/

Unsupported System Vectors

Patching is an important prevention tool

Unsupported systems aren’t patched

Outdated operating systems will eventually not even be supported by the manufacturer

A single system could be an entry - keep inventory and records current

Unsecure Network Vectors

The network connects everything

Ease of access for the attackers

View all (non-encrypted) data

If you’re using Wireless, ensure not using outdated/insecure protocols like WEP, WPA, WPA2

Open Service Ports

Most network-based services connect over a TCP or UDP port

An “open” port

Default Credentials

Most devices have default usernames and passwords - did you change yours?

https://www.routerpasswords.com/

Supply Chain Vector

Tamper with the underlying infrastructure or manufacturing process
