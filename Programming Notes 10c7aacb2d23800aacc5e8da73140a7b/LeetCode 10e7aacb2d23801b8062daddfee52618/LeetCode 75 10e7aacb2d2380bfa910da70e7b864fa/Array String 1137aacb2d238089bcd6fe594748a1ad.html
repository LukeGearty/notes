Merge Two String Alternately

You are given two strings word1 and word2. Merge the strings by adding letters in alternating order, starting with word1. If a string is longer than the other, append the additional letters onto the end of the merged string.

Return the merged string.

Example 1:

Input: word1 = "abc", word2 = "pqr"
Output: "apbqcr"
Explanation: The merged string will be merged as so:
word1:  a   b   c
word2:    p   q   r
merged: a p b q c r

Example 2:

Input: word1 = "ab", word2 = "pqrs"
Output: "apbqrs"
Explanation: Notice that as word2 is longer, "rs" is appended to the end.
word1:  a   b
word2:    p   q   r   s
merged: a p b q   r   s

Example 3:

Input: word1 = "abcd", word2 = "pq"
Output: "apbqcd"
Explanation: Notice that as word1 is longer, "cd" is appended to the end.
word1:  a   b   c   d
word2:    p   q
merged: a p b q c   d

Constraints:

  • 1 <= word1.length, word2.length <= 100
  • word1 and word2 consist of lowercase English letters.

My Solution:

set an integer variable to 0, and string variable to be empty

while that integer is less than the length of word1 or less than length of word2 - boolean logic, so that both conditions need to be false so the entire statement is false. So if one condition is false, the while loop will still keep going.
if i is less than the length of word1, add word1 at index of i to the result
If i is less than the length of word2, add word2 at index of i to the result
Do not use else if, use if to keep it alternating

Increment i on each round

Return result
Time Complexity: O(N)

Kids With the Greatest Number of Candies

There are n kids with candies. You are given an integer array candies, where each candies[i] represents the number of candies the ith kid has, and an integer extraCandies, denoting the number of extra candies that you have.

Return a boolean array result of length n, where result[i] is true if, after giving the ith kid all the extraCandies, they will have the greatest number of candies among all the kids, or false otherwise.

Note that multiple kids can have the greatest number of candies.

Example 1:

Input: candies = [2,3,5,1,3], extraCandies = 3
Output: [true,true,true,false,true]
Explanation: If you give all extraCandies to:
- Kid 1, they will have 2 + 3 = 5 candies, which is the greatest among the kids.
- Kid 2, they will have 3 + 3 = 6 candies, which is the greatest among the kids.
- Kid 3, they will have 5 + 3 = 8 candies, which is the greatest among the kids.
- Kid 4, they will have 1 + 3 = 4 candies, which is not the greatest among the kids.
- Kid 5, they will have 3 + 3 = 6 candies, which is the greatest among the kids.

Example 2:

Input: candies = [4,2,1,1,2], extraCandies = 1
Output: [true,false,false,false,false]
Explanation: There is only 1 extra candy.
Kid 1 will always have the greatest number of candies, even if a different kid is given the extra candy.

Example 3:

Input: candies = [12,1,12], extraCandies = 10
Output: [true,false,true]

My Solution:
first, set a result vector/array

second, find the greatest amount of candies and set it to a variable

third, loop through candies array again, and compare the candies at index i + extraCandies to greatest. If it is greater or equal to greatest, add a true to the result. If not, add a false.

Return result vectorapp

Time Complexity: O(n)

Example 1:

Input: flowerbed = [1,0,0,0,1], n = 1

Example 2:

Input: flowerbed = [1,0,0,0,1], n = 2
Output: false

Constraints:

  • flowerbed[i] is 0 or 1.

Reverse Vowels of a String

Given a string s, reverse only all the vowels in the string and return it.

The vowels are 'a', 'e', 'i', 'o', and 'u', and they can appear in both lower and upper cases, more than once.

Example 1:

Input: s = "IceCreAm"

Output: "AceCreIm"

Explanation:

The vowels in s are ['I', 'e', 'e', 'A']. On reversing the vowels, s becomes "AceCreIm".

Example 2:

Input: s = "leetcode"

Output: "leotcede"

Constraints:

  • 1 <= s.length <= 3 * 105
  • s consist of printable ASCII characters.

My solution:

Initialize an array of vowels uppercase and lowercase

Initialize an array to store the vowels in the string, vowels_in_string

Initialize an integer variable to be the length of the input string, length

Initialize the empty string result

Loop through the string:

If the current index is a vowel, add it to the list of vowels in the string

end loop

Initialize an index variable to be the length of the vowels_in_string array

Loop from 0 to length - 1:

If s[i] is a vowel, result adds vowels_in_string[index], decrement index

else, result adds s[i]

End loop

return result

Time Complexity: O(n)

class Solution {
public:
    string reverseVowels(string s) {
        int length = s.length();
        string vowels = "";
        string result = "";
        for (int i = 0; i < length; i++) {
            if (s[i] == 'A' || s[i] == 'E' || s[i] == 'I' || s[i] == 'O' || s[i] == 'U' || s[i] == 'a' || s[i] == 'e' || s[i] == 'i'|| s[i] == 'o' || s[i] == 'u') {
                vowels+=s[i];
            }
    }

    int indexVowels = vowels.length() - 1;

    for (int i = 0; i < length; i++) {
        if (s[i] == 'A' || s[i] == 'E' || s[i] == 'I' || s[i] == 'O' || s[i] == 'U' || s[i] == 'a' || s[i] == 'e' || s[i] == 'i'|| s[i] == 'o' || s[i] == 'u') {
            result+=vowels[indexVowels--];
        }
        else {
            result+=s[i];
        }
    
    }

    return result;
    }
};
